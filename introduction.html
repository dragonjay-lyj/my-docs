                                  # Requirements

                                  Installing Hexo is quite easy and only requires the following beforehand:

                                  • Node.jsopen in new window (Should be at least Node.js 10.13, recommends 18.0 or higher)
                                  • Gitopen in new window
                                  • Visual Studio Codeopen in new window(Optionally, you can also edit directly on Github)

                                  open in new windowopen in new windowopen in new window

                                  # Install Git

                                  • Windows: Download & install gitopen in new window.
                                  • Mac: Install it with Homebrewopen in new window, MacPortsopen in new window or installeropen in new window.
                                  • Linux (Ubuntu, Debian): sudo apt-get install git-core
                                  • Linux (Fedora, Red Hat, CentOS): sudo yum install git-core

                                  For Mac users

                                  You may encounter some problems when compiling. Please install Xcode from App Store first. After Xcode is installed, open Xcode and go to Preferences -> Download -> Command Line Tools -> Install to install command line tools.

                                  # Install Node.js

                                  Node.js provides official installeropen in new window for most platforms.

                                  Alternative installation methods:

                                  • Windows: Install it with nvsopen in new window (recommended) or nvmopen in new window.
                                  • Mac: Install it with Homebrewopen in new window or MacPortsopen in new window.
                                  • Linux (DEB/RPM-based): Install it with NodeSourceopen in new window.
                                  • Others: Install it through respective package manager. Refer to the guideopen in new window provided by Node.js.

                                  nvs is also recommended for Mac and Linux to avoid possible permission issue.

                                  Windows

                                  If you use the official installer, make sure Add to PATH is checked (itâ€™s checked by default).

                                  Mac / Linux

                                  If you encounter EACCES permission error when trying to install Hexo, please follow the workaroundopen in new window provided by npmjs; overriding with root/sudo is highly discouraged.

                                  Linux

                                  If you installed Node.js using Snap, you may need to manually run npm install in the target folder when initializingopen in new window a blog.

                                  # Install Hexo

                                  Once all the requirements are installed, you can install Hexo with npm:

                                  $ npm install -g hexo-cli

                                  # Advanced installation and usage

                                  Advanced users may prefer to install and use hexo package instead.

                                  $ npm install hexo

                                  Once installed, you can run Hexo in two ways:

                                  1. npx hexo <command>
                                  2. Linux users can set relative path of node_modules/ folder:

                                  echo 'PATH="$PATH:./node_modules/.bin"' >> ~/.profile

                                  then run Hexo using hexo <command>

                                  Once Hexo is installed, run the following commands to initialize Hexo in the target <folder>.

                                  $ hexo init foldername $ cd foldername
                                  $ npm install
